Released April 3, 2017.

In fixing a riak and riak-admin issue, we inadvertently used a Bash-specific script variable ($RANDOM), causing the riak attach and riak top commands to fail on Ubuntu when /bin/sh is aliased to Dash.

Bugs Fixed

  • [riak PR 909 & node_package PR 217] Replace $RANDOM with PID to support non-Bash.
